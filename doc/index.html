Pro Wrestling Superstar

Pro Wrestling Superstar is a professional wrestling game/simulation based on the Superstar Pro Wrestling card and dice game from the mid 1980's. Pro Wrestling Superstar captures the excitement and action of professional wrestling. Additionally, PWS makes a return to “good old fashioned” gaming. Roll the dice and see what happens...

Features

  • PWS provides a simple, but powerful, user interface for gameplay

  • Multiplayer gaming, so you can play with others across the Internet

  • Fast paced and easy to learn gameplay

  • Open Architecture. The source code is provided as well as tools for adding matches, wrestlers, cards, etc..

  • Ability to create federations and wrestler track wins/losses

  • Doesn't require high end hardware to run

  • A return to old fashioned RDASH (Roll the Dice And See what Happens) gameplay. ;)
